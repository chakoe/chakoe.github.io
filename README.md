# 赶圩日历

一个简单易用的赶圩日历网页应用，用于查询特定地区的圩日信息。

## 功能特性

- **自动回到当日**：页面刷新后自动显示当前日期的圩日信息
- **日期选择**：支持通过日期选择器查看任意日期的圩日信息
- **实时显示**：显示当前日期的圩日名称和状态
- **下圩日提示**：显示距离下一个圩日的天数和具体日期
- **历史记录**：展示最近30天的圩日历史记录
- **响应式设计**：适配不同屏幕尺寸的设备

## 技术实现

- **前端技术**：HTML5, CSS3, JavaScript (ES6+)
- **日期处理**：统一使用本地时间，避免时区问题
- **页面设计**：现代化的渐变背景和流畅的动画效果
- **性能优化**：减少重复的日期对象创建，提高效率

## 使用方法

1. 直接打开 `index.html` 文件即可使用
2. 默认显示当前日期的圩日信息
3. 点击日期选择器可以查看其他日期的圩日信息
4. 页面刷新后会自动回到当前日期

## 圩日规则

本日历基于以下规则计算圩日：

- **基准日期**：2025年11月12日（太守圩日）
- **轮换周期**：每3天一轮换，依次为：
  - 太守
  - 新桥
  - 芦圩
