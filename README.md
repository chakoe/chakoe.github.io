# 圩日日历应用

## 项目简介

这是一个基于网页的圩日日历应用，用于显示和计算中国农村地区传统的圩日（集市日）信息。应用采用"三天为一圩"的周期计算方式，支持三个圩日名称：太守、新桥、芦圩。

## 功能特性

### 核心功能
- **当前日期显示**：显示当前日期和星期信息
- **圩日查询**：查询任意日期的圩日信息
- **日期选择器**：支持手动选择日期进行查询
- **下一个圩日**：显示距离下一个圩日的天数和具体日期
- **历史记录**：显示过去30天的圩日历史记录

### 辅助功能
- **本地存储**：保存用户上次选择的日期
- **自动更新**：每分钟自动更新当前日期的显示
- **响应式设计**：适配不同屏幕尺寸的设备
- **交互效果**：丰富的动画和过渡效果，提升用户体验

## 使用方法

1. **查看当前圩日**：打开应用即可看到当前日期的圩日信息
2. **查询历史圩日**：在日期选择器中选择需要查询的日期
3. **查看历史记录**：滚动查看历史圩日记录列表

## 圩日规则

应用采用以下圩日规则：
- 以2025年11月12日（UTC时间）为基准日
- 每三天为一个周期，依次为：太守 → 新桥 → 芦圩 → 太守 → ...
- 所有日期都是圩日，只是不同日期对应不同的圩日名称

## 技术实现

### 前端技术
- **HTML5**：页面结构和语义化标记
- **CSS3**：样式设计和响应式布局
- **JavaScript (ES6+)**：核心功能实现和交互逻辑

### 核心功能实现

1. **圩日计算**：
   - 使用UTC时间进行日期计算，避免时区问题
   - 通过基准日和天数差计算圩日名称

2. **日期处理**：
   - 封装日期处理工具函数，确保日期格式一致性
   - 支持日期字符串和日期对象的相互转换

3. **数据持久化**：
   - 使用localStorage保存用户选择的日期
   - 页面加载时自动恢复上次选择的日期

4. **用户界面**：
   - 现代化的渐变背景和卡片式布局
   - 平滑的过渡动画和悬停效果
   - 自定义滚动条样式

## 开发说明

### 配置修改

在`今日圩日.html`文件中，可以修改以下配置：

```javascript
const CONFIG = {
    baseDate: new Date(Date.UTC(2025, 10, 12)), // 太守基准日（使用UTC时间）
    marketNames: ['太守', '新桥', '芦圩'],      // 圩日名称列表
    defaultHistoryDays: 30                       // 默认显示的历史天数
};
```
